# Ephone-S · 小手机（Android 版）

> 基于 **EE** 的 Ephone / 小手机，以及 **JCY、330、kuku** 等创作者版本，二次改编的 Android 原生实现。
> 一款模拟手机桌面的酒馆（SillyTavern）衍生陪伴应用：在小手机里选择并使用各类应用，与 AI 联系人对话。

## 📱 功能简介

小手机模拟一台手机，提供桌面、相册、购物、设置等模块，并以 AI 为联系人进行对话。
内置「酒馆」入口仅作为**辅助玩酒馆的工具**，通过 WebView 运行独立的 SillyTavern，本项目**未修改酒馆源代码**。

主要模块：桌面、AI 对话、相册、购物商城、健康、设置，以及酒馆辅助入口等。
